## Run Locally

Clone the project

```bash
  git clone git@github.com:vineet98-code/NINEGAP.git
```

Navigate to the project

```bash
  cd .\NINEGAP
```

Install dependencies

```bash
  npm install
```

view client in the browser

Local: http://localhost:3000
